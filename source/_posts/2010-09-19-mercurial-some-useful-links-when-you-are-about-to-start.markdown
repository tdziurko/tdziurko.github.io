In my current project for a first time I am working with [Mercurial](http://mercurial.selenic.com/) version control system. And here are some links, materials and tutorials which I found very useful at the beginning of work with this distributed source code management so different than CVS or SVN.



	
  1. [http://hginit.com/top/](http://hginit.com/top/) - Short but very concise tutorial written by Joel Spolsky, yeah [that](http://www.joelonsoftware.com/) Joel :) All written in a very entertaining way

	
  2. [TortoiseHg](http://tortoisehg.bitbucket.org/) - popular application known from its versions for SVN/CVS with version for Mercurial

	
  3. [Zdalne repozytorium za darmo z użyciem serwisu Dropbox](http://www.h4ck3r.net/2010/05/11/mercurial-hg-with-dropbox/) - Article explaining how to connect Mercurial with Dropbox to end with free self-backuping repository.

	
  4. [MercurialEclipse](http://javaforge.com/project/HGE) - Eclipse plugin allowing to work with Mercurial from this IDE

	
  5. [Mercurial - the Definitive Guide](http://hgbook.red-bean.com/) - Book explaining almost every detail of Mercurial

	
  6. [Mercurial by Example (PDF)](http://www.jemander.se/MercurialByExample.pdf) - Mini article showing how to use Mercurial with easy to follow examples

	
  7. [Branch i merge w Mercurial](http://stevelosh.com/blog/2009/08/a-guide-to-branching-in-mercurial/) - Guide how to create branches, how to merge and how spend only a few moments (in contrary to SVN) on merging two distant branches

	
  8. [Development workflow for Mercurial](http://blogs.techrepublic.com.com/programming-and-development/?p=3237) - article at TechRepublic


And as always, the latest list of articles and tutorials can be found on the Wiki site - [http://mercurial.selenic.com/wiki](http://mercurial.selenic.com/wiki).
