Inspired by posts with useful links at [blog by Piotr Paradziński](http://it-researches.blogspot.com/) I decided to create similar one with package containing interesting materials about Wicket and accompanying libraries. I am going to update this post in the future as new things appear.


### Most important links





	
  1. [Project official website](http://wicket.apache.org/)

	
  2. Very active and very helpful [mailing list](http://old.nabble.com/Wicket---User-f25133.html)

	
  3. Live examples of components usage - [link](http://www.wicketstuff.org/wicket14/) i [mirror](http://wicketbyexample.com/examples/)

	
  4. [Project Wiki](http://cwiki.apache.org/WICKET/)

	
  5. [Dzone Refcard about Apache Wicket](http://refcardz.dzone.com/refcardz/getting-started-apache-wicket)

	
  6. [Javadocs of Apache Wicket projects](http://wicketbyexample.com/api) - javadocs of Wicket, Wicket-extensions and some other Wicket libraries in one place

	
  7. [Latest Javadocs](http://wicket.apache.org/apidocs/1.4/index.html?overview-summary.html)  for 1.4.15 Wicket version




### Wicket extras, add-on libraries and tools





	
  1. [wiQuery](http://code.google.com/p/wiquery/) - jQuery integration with Wicket

	
  2. [WicketStuff](https://github.com/wicketstuff/core/wiki) - many different components (integrations with various JavaScript libraries, additional classes and projects extending capabilities of Wicket) with examples

	
  3. [WicketCool](http://code.google.com/p/wicketcool/) - project by [Paul Szulc](http://paulszulc.com/) helping with rapid application development using Wicket, Spring 3 and JPA2.

	
  4. [Maven archetypes for Wicket](http://jweekend.co.uk/dev/LegUp) by jWeekend  - project starters with many different technologies: JDBC, JPA, Spring 3.0, Hibernate, Guice or WarpPersist

	
  5. [Enhanced Wicket Tester](http://code.google.com/p/enhancedwickettester/) - improved version of WicketTester project making testing Wicket app much easier, also from [Paul Szulc](http://paulszulc.com/).

	
  6. [Brix](http://code.google.com/p/brix-cms/), CMS using Wicket

	
  7. [Visural Wicket](http://code.google.com/p/visural-wicket/) - set of very attractive, interactive and useful Wicket components ([live demo](http://visural-wicket-examples.appspot.com/))

	
  8. [Wicketopia](http://wicketopia.sourceforge.net/) - Rapid Application Development (RAD) library for Wicket framework

	
  9. [FiftyFiveWicket](http://opensource.55minutes.com/trac/wiki/FiftyFiveWicket) - set of tools increasing productivity of development process using Wicket (helper methods, wrappers for integration with Spring, etc.)
